# Digit Recognizer - Gradient Boosting Classifier

A simple web app built with *Streamlit* that predicts handwritten digits (0-9) using a *Gradient Boosting Classifier* trained on the [MNIST-like Digits dataset](https://scikit-learn.org/stable/modules/generated/sklearn.datasets.load_digits.html) from scikit-learn.

---

## Features

- Predict handwritten digits based on 64 pixel values (0-16).  
- Uses *Gradient Boosting Classifier* for high accuracy.  
- Displays model accuracy on the test set.  
- Saves and loads the trained model using *pickle*.
